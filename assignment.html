Differentacademic disciplines require different

writing format styles . There are several formatting styles,Some important text but the three most common that are used in academic writing are the American Psychological Association (APA), the Modern Language Association (MLA), and the Chicago Manual of Style (CMS). Which style of formatting is required depends on the topic of the paper and the discipline or academic field. All three of these styles have some basic formatting guidelines in cecommon. Within all formatting styles papers should be double-spaced and should have one-inch margins throughout. All styles also require a basic typeface such as Times New Roman and a readable font size, usually 12 pt. The styles differ, however, with regards to cover pages, headers, and citations.

The Main Languages of the Web

HTML is the standard markup language for creating Web pages. HTML describes the structure of a Web page, and consists of a series of elements. HTML elements tell the browser how to display the content.

CSS is a language that describes how HTML elements are to be displayed on screen, paper, or in other media. CSS saves a lot of work, because it can control the layout of multiple web pages all at once.

JavaScript is the programming language of HTML and the Web. JavaScript can change HTML content and attribute values. JavaScript can change CSS. JavaScript can hide and show HTML elements, and more.
